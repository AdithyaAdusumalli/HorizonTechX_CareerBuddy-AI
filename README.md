# 🎓 CareerBuddy-AI

A smart, console-based career guidance chatbot built using Python.

CareerBuddy-AI is designed to assist students with academics, placements, internships, resume preparation, interview readiness, hackathon participation, and personal motivation. It combines multiple chatbot styles into a single application, providing an interactive and helpful experience for students and fresh graduates.

---

## 🚀 Project Overview

CareerBuddy-AI is an enhanced rule-based chatbot that acts as a virtual career companion.

Unlike a basic chatbot that only responds to greetings, CareerBuddy AI Pro provides:

* Academic guidance
* Placement preparation tips
* Resume suggestions
* Interview advice
* Python learning guidance
* GitHub best practices
* Hackathon recommendations
* Motivation and productivity support

The chatbot also maintains chat history and includes utility features such as date, time, jokes, and inspirational quotes.
